    Vue.js 入门指南：用组合式 API 构建现代前端

    告别冗长的 Options API，Vue 3 的组合式 API 让逻辑复用更简单、代码组织更清晰。
    无论你是从 React 转来，还是前端新手，都能快速上手并提升开发效率。

const count = ref(0)        // 响应式基本类型
const state = reactive({    // 响应式对象
